# DB backup tools:
-------------------
1) For Postgres DB:
   Barman (Backup and Recovery Manager)
   - Enterprise-grade backup solution
   - Native S3 support via boto3
   - Incremental backups
   - Retention policy management
   - Monitoring and alerting
2) For Cassandra DB:
   - Cassandra Native Snapshots + S3 Upload
   - Medusa (Spotify's Cassandra Backup Tool)
   - Tablesnap
   - Apache Cassandra Backup (ACB)
   - Instaclustr Cassandra Backup
   - Custom Scripts + S3 Integration
